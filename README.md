## 1. DynaTTA
- **论文链接**：[DynaTTA (OpenReview)](https://openreview.net/pdf?id=a399SmgWGl)
- **代码仓库**：[shivamgrover/DynaTTA](https://github.com/shivam-grover/DynaTTA)

## 2. PETSA

- **代码仓库**：[BorealisAI/PETSA](https://github.com/BorealisAI/PETSA)

## 3. TAFAS
- **论文链接**：[TAFAS (arXiv)](https://arxiv.org/abs/2501.04970)
- **代码仓库**：[kimanki/TAFAS](https://github.com/kimanki/TAFAS)
